# My Portfolio

This project serves as my personal portfolio, showcasing my skills and projects. It's built using Next.js, React, Tailwind CSS, and shadcn-ui, demonstrating my proficiency in modern web development technologies.


## Technologies Used

*   [Next.js](https://nextjs.org/)
*   [React](https://reactjs.org/)
*   [Tailwind CSS](https://tailwindcss.com/)
*   [shadcn-ui](https://ui.shadcn.com/)
*   [TypeScript](https://www.typescriptlang.org/)
